# Monster Factory

### 여행에서 만나는 또 다른 나

여행은 새로운 장소를 보는 일이 아니라, 다양한 경험을 통해 세상과 나를 알아가는 과정입니다.

Monster Factory는 여행을 통해 새로운 가능성을 발견하고, 익숙한 자신을 넘어 또 다른 나를 만나도록 돕는 여행 라이프스타일 브랜드입니다.

---

## 1. 핵심 가치

### Explore Beyond

익숙한 일상을 벗어나 새로운 경험에 도전합니다.

### Discover Yourself

여행 속에서 몰랐던 취향과 가능성을 발견합니다.

### Express Individuality

정형화된 여행보다 자신만의 방식과 개성을 존중합니다.

### Enjoy the Journey

목적지보다 여행 과정 자체의 즐거움을 중요하게 생각합니다.

---

## 2. 차별점

### 개성을 위한 여행용품

수납과 기능만을 위한 제품이 아닌, 사용자의 취향과 정체성을 표현하는 여행 아이템을 제안합니다.

### 경험 중심 브랜드

제품 판매에 그치지 않고 여행 이야기, 문화, 라이프스타일을 함께 제안합니다.

### 나를 발견하는 여행

어디를 가는지보다 어떤 경험을 하는지에 집중합니다.

### 독창적인 디자인

획일적인 여행용품 대신 개성과 스토리가 담긴 디자인을 제공합니다.

---

## 3. 타겟 사용자

### Primary Target

25~45세

여행을 단순한 휴식이 아닌 새로운 경험과 자기 발견의 기회로 생각하는 사람

- 혼자 여행을 즐기는 사람
- 새로운 문화와 취향을 경험하는 사람
- 개성 있는 브랜드를 선호하는 사람
- SNS에 자신의 여행 경험을 기록하는 사람

### Secondary Target

- 디지털 노마드
- 워케이션 사용자
- 라이프스타일 브랜드 소비층
- 취향 소비를 중시하는 MZ 및 영 프로페셔널

Image Overlay System

1. Giant Background Type

이미지 뒤 또는 이미지 위에 아주 큰 영문 텍스트를 배치한다.

예시:

- TOKYO
- PORTO
- BUSAN
- RAIN
- WORK
- COLOR
- GET LOST

규칙:

- 화면 밖으로 일부 잘려도 됨
- opacity는 낮게 사용
- 제품이나 인물을 가리지 않음
- 잡지 커버 타이틀처럼 크게 사용
- --

2. Monster Sticker

사진 위에 실제 스티커처럼 붙이는 그래픽 요소.

문구 예시:

- GET LOST
- PACK LIGHT
- WEEKEND ONLY
- RAIN APPROVED
- NOT FOR PLANNERS
- NEW CITY
- FIND YOUR MONSTER

형태:

- 원형
- 별형
- 구름형
- 말풍선형
- 사각 라벨형

규칙:

- 섹션당 1~2개만 사용
- 약간 기울여 배치
- 형광 옐로우, 핑크, 애시드 그린 사용
- 너무 많이 쓰지 않음
- --

3. Chat Bubble

여행 중 주고받은 메시지처럼 보이는 작은 말풍선.

예시:

- “pics or it didn’t happen”
- “lost again”
- “good coffee ahead”
- “which color today?”
- “one more email”
- “still raining”

스타일:

- 화이트 배경
- 블랙 라인
- 작은 텍스트
- 형광 옐로우 또는 핑크 그림자

사용 위치:

- City Snap 이미지
- Special Exhibition 이미지
- Hero 이미지 일부
- --

4. Ticket Sticker

도시, 시간, 장소 정보를 티켓처럼 표시한다.

예시:

- TOKYO / 07:42 AM
- BANGKOK / 04:18 PM
- PORTO / 06:05 PM
- BUSAN / 11:36 PM

스타일:

- 작은 티켓 라벨
- 모서리가 약간 둥근 사각형
- 흰 배경 또는 형광 배경
- 검은 텍스트
- --

5. Marker Highlight

타이틀이나 짧은 문구 뒤에 형광펜으로 그은 듯한 형태를 사용한다.

예시:

- 형광 옐로우 중심 사용

#최종 결과물

HTML, CSS, JS로 Monster Factory 브랜드 랜딩페이지를 만들어줘.

목표:

- 단순 쇼핑몰이 아니라, 여행 매거진 + 트래블 라이프스타일 브랜드 느낌
- 기본은 정돈된 Editorial Grid
- 그 위에 형광 스티커, 말풍선, 낙서형 아이콘 같은 playful 요소를 10% 정도만 추가
- 전체 무드는 90% 질서 / 10% 장난
- 로고는 블랙 텍스트 기반
- 스포티하고 캐주얼하지만 너무 유치하지 않게

기술 조건:

- 파일은 index.html, style.css, script.js 3개로 구성
- 외부 라이브러리 사용하지 않기
- 반응형 대응
- Desktop 기준 최대 너비 1440px
- Mobile 390px 기준 대응
- 실제 이미지는 없으므로 이미지 영역은 감각적인 placeholder block으로 구현
- placeholder에는 이미지 콘셉트 텍스트를 작게 넣기

비주얼 시스템:

- Background: #F7F5EF 또는 #FFFFFF
- Text: #111111
- Sub Text: #666666
- Accent Red: #E7352C
- Neon Yellow: #EFFF1A
- Neon Pink: #FF4FB8
- Acid Green: #B8FF2C
- Typography: bold sans-serif 중심
- Hero Title은 매우 크게
- Section Title은 잡지 타이틀처럼 강하게
- Body는 정돈되고 읽기 쉽게

공통 컴포넌트:

1. Header
- 좌측 MONSTER FACTORY
- 우측 Shop / Stories / Snap / Join
- sticky header
- 흰 배경, 블랙 텍스트
1. Sticker Component
- 원형, 별형, 말풍선, 테이프형 라벨
- 문구 예시: GET LOST, PACK LIGHT, WEEKEND ONLY, NOT FOR PLANNERS, LOST AGAIN
- 섹션마다 1~2개만 사용
- position absolute로 사진이나 타이틀 근처에 붙이기
1. CTA Button
- 블랙 배경 / 흰 글자
- hover 시 neon yellow 배경 / 블랙 글자

중요:

- 잡지 편집 디자인처럼 만들어줘.
- 카드와 섹션은 정돈되어야 하지만, 스티커와 라벨은 약간 어긋나게 배치해줘.
- 전체 페이지가 “매거진 감성 + 여행 스티커 문화”처럼 보이게 해줘.
